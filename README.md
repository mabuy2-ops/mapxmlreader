# 地籍図XML → DXF 変換アプリ

法務省地籍図XMLファイルをCAD用DXFファイルに変換するアプリケーションです。

---

## 概要

法務省が配布する地籍図XMLフォーマット（`http://www.moj.go.jp/MINJI/tizuxml`）を読み込み、JW-CADやAutoCADで開けるDXFファイルに変換します。

- ZIPファイルまたはXMLファイルをブラウザにドロップするだけで変換できます
- 座標点・筆界線・地番・基準点・図郭枠を自動的にレイヤー分けして出力します
- 座標系は公共座標系（X=北, Y=東）からDXF標準（X=東, Y=北）に自動変換します

**2通りの使い方があります：**

| 方式 | ファイル | Pythonが必要か | インターネットが必要か |
|------|----------|----------------|----------------------|
| **スタンドアロン（推奨）** | `index_standalone.html` | 不要 | JSZip取得のため初回のみ必要 |
| **Flaskサーバー** | `app.py` | 必要 | 不要 |

## A. スタンドアロン版（サーバー不要）

### 起動方法

`index_standalone.html` をダブルクリックしてブラウザで開くだけです。

```
G:\マイドライブ\※VSCode\mapxmlreader\index_standalone.html
```

> インターネット接続が必要です（JSZipライブラリをCDNから自動取得します）。

### 仕組み

- ZIP展開: **JSZip**（CDN）
- XML解析・DXF生成: ブラウザ内の **Web Worker**（UIをブロックしない）
- ファイルのやり取りはすべてブラウザ内で完結（データはサーバーに送信されません）

### 動作環境

| 項目 | 内容 |
|------|------|
| ブラウザ | Chrome 90以上 / Edge 90以上 / Firefox 84以上 / Safari 15.4以上 |
| Python | **不要** |

## 使い方（両版共通）

1. ブラウザでアプリを開く
2. ZIPファイル（または展開済みXMLファイル）をドロップゾーンにドラッグ＆ドロップ、またはクリックして選択
3. **「変換してDXFをダウンロード」** ボタンを押す
4. 変換完了後、自動的にDXFファイルがダウンロードされます

変換には概ね **10〜30秒程度** かかります（ファイルサイズ・PC性能による）。

## 仕様

### 入力XMLの構造（法務省地籍図XMLフォーマット）

```
<地図>
  <空間属性>
    <zmn:GM_Point>   … 座標点（公共座標系 X=北, Y=東）
    <zmn:GM_Curve>   … 線分（直接座標 or 点IDへの参照）
    <zmn:GM_Surface> … 面（曲線IDの参照リスト）
  </空間属性>
  <主題属性>
    <筆>       … 土地区画（大字名・地番・精度区分・面IDへの参照）
    <筆界線>   … 境界線（曲線IDへの参照・線種別）
    <筆界点>   … 境界点（点IDへの参照・点番名）
    <基準点>   … 基準点（点IDへの参照・名称・種別）
  </主題属性>
  <図郭>       … 図郭枠（地図番号・縮尺・四隅座標）
</地図>
```

### 座標変換

公共座標系では X が北方向、Y が東方向のため、DXF出力時に入れ替えを行います。

```
DXF_X = 公共座標_Y（東方向）
DXF_Y = 公共座標_X（北方向）
```

### 出力DXFレイヤー構成

| レイヤー名（ASCII） | 意味 | 色 | エンティティ |
|---------------------|------|----|-------------|
| `BOUNDARY`   | 筆界線 | 赤（1） | LINE |
| `LANDNUMBER` | 地番テキスト | 緑（3） | TEXT |
| `BND_POINT`  | 筆界点 | 青（5） | POINT |
| `REF_POINT`  | 基準点・多角点 | 黄（2） | POINT / TEXT |
| `FRAME`      | 図郭枠・地図番号 | 白（7） | LINE / TEXT |

> **レイヤー名をASCIIにしている理由**  
> JW-CADはDXFのUTF-8日本語レイヤー名を正しく認識できないため、ASCII英字のレイヤー名を使用しています。

### DXFエンティティについて

LWPOLYLINE（軽量ポリライン）はJW-CADで表示されないため、すべての線分を **LINE エンティティ** として出力しています。

### テキスト高さ

図郭の縮尺分母から自動算出します。

```
テキスト高さ = 縮尺分母 × 0.003 [m]
```

例: 縮尺 1/500 → テキスト高さ 1.5m（印刷時 3mm に相当）

## 注意事項

- JW-CADで開く際、座標値が大きいため最初は全体表示（`Ctrl+F`）で確認してください。
- スタンドアロン版はブラウザのセキュリティ制限により、一部の古いブラウザでは動作しない場合があります。
- Flaskサーバー版は開発用サーバーです。本番運用には Gunicorn などのWSGIサーバーをご使用ください。
